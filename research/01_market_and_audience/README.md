# 📂 Рынки и аудитория

**Эта папка — для ответственного за анализ рынков и аудитории.**

Что здесь: кто наша аудитория, что у неё болит, кто конкуренты, какие сегменты бить первыми и какие рынки/языки брать.

| Файл | Содержимое |
|---|---|
| `01_audience_profile.md` | Демография, психография, JTBD, 2 синтезированные персоны (Линда 49 / Дэвид 52) |
| `02_pains_and_needs.md` | 13 ранжированных болей с цитатами (EN-оригинал + RU), неудовлетворённые потребности, VOC-словарь |
| `03_competitors_market.md` | 12 книг-конкурентов (англорынок), 8 white space, размер/цены рынка |
| `05_segmentation_priorities.md` | 4 сегмента, матрица приоритизации, рекомендация по первому удару |
| `06_new_markets_scan.md` | Ранжирование новых рынков под перевод (исп./нем./UK/CA/AU…) |
| `07_market_ukraine.md` | Украинский рынок: размер, культура, конкуренты, запросы, VOC |
| `08_market_russophone_diaspora.md` | Русскоязычная диаспора (не РФ): каналы, конкуренты, запросы |
| `10_market_spanish_deep.md` | Испаноязычный рынок (углублённо): рынок ES/LatAm/US-Hispanic, боли (ES+RU), конкуренты, gordofobia, сегментация |

**Главное:** гипотеза автора — гео США ✅, возраст 35+ ✅ (усилено), боли 🔶 скорректированы (BED №1, не булимия; расширить на субклиническое переедание). Первый удар — сегмент «измученные циклом 35–55»; мужская линия 40+ — бренд-дифференциатор. Детали и вердикт — в `../00_executive_summary.md`.

---

**📖 Термины:** **РПП** — расстройство пищевого поведения; **BED** — компульсивное переедание (самое частое РПП); **disordered eating** — нарушенное пищевое поведение без диагноза; **JTBD** — «работа», которую нанимают книгу выполнить; **VOC** — голос клиента (дословный язык аудитории); **персона** — портрет сегмента из данных; **white space** — незанятый рыночный пробел. (Полные глоссарии — в каждом файле папки.)
