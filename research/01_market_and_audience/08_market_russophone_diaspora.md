# 08 — Рынок русскоязычной диаспоры (НЕ РФ)

**Дата:** 2026-05-26 · **Папка:** `01_market_and_audience`
**Фокус:** русскоязычные ЗА ПРЕДЕЛАМИ РФ (Германия, Израиль, США, Канада, Прибалтика, волна-2022). РФ исключена осознанно — платёжные/дистрибуционные барьеры.
**Полная база (профили конкурентов, все VOC-цитаты, источники):** `../raw/08_market_russophone_diaspora.md`.

## ВЫВОДЫ ПЕРВЫМ ДЕЛОМ

1. **Адресуемый рынок — ориентировочно 6–9 млн человек.** Центры тяжести: Германия (3–5 млн русскоязычных), Израиль (~1.1–1.3 млн, 15–20% страны), США (~1 млн), Канада (~0.5 млн), Прибалтика (~0.45 млн) + волна-2022 (~0.7–1 млн, часто IT с высоким доходом и острым запросом на психологическое восстановление). *(Оценки по языковому критерию, порядок величины.)*
2. **Платёжеспособность высокая там, где надо** — Израиль и США/Канада (книга $12–20 = незаметная трата). Это не «бедная диаспора».
3. **Главный конкурент — Светлана Бронникова** (книга «Интуитивное питание» + экосистема IntuEat, сама живёт в Нидерландах = часть диаспоры). Сильна авторитетом и системой, но **имеет связку незакрытых пробелов, которые «Новый Код» закрывает разом.**
4. **Мужской голос + личный опыт РПП — структурный разрыв рынка.** Ни одна из 8+ русскоязычных книг ниши не написана мужчиной с личным опытом анорексии. Запросы «анорексия у мужчин», «РПП у мужчин» — почти без качественного контента.
5. **Дистрибуция решаема без РФ:** Amazon Kindle KDP + прямые продажи через Telegram + Gumroad/Payhip (Stripe/PayPal). **LitRes для диаспоры не годится** (оплата иностранными картами заблокирована).

### Бронникова — главные пробелы (= вход для «Нового Кода»)
- **Нет личной истории** (клиницист-наблюдатель «сверху»). · **Нет мужского голоса.** · **Структурный хаос** (частая жалоба: «непонятно, с чего начать»). · **«Не для уже нарушенного ПП».** · Фокус почти только на еде — **нет линий движения и психики как отдельных компонентов.**

> «В голове остается хаос, нет ощущения логического повествования» — Natasha Alekseeva (LitRes)
> «трудно понять, в какой последовательности делать… с чего вообще начать» — margarita.mozhaeva (LitRes)
> «Не может интуитивно хорошо питаться тот, у кого уже нарушено пищевое поведение» — Foxie Fox (LitRes)
> «Я поправилась на 10 кило читая эту книгу» — Зу (LitRes)

## ДИСТРИБУЦИЯ ДЛЯ ДИАСПОРЫ (без РФ)

1. **Amazon Kindle KDP** (через Ridero или напрямую) — главный; работает во всех целевых странах, любая карта, русскоязычный раздел существует. Роялти ~70%.
2. **Telegram + Gumroad/Payhip** — прямые продажи PDF/epub, Stripe/PayPal, ~95% роялти, без гео-ограничений оплаты.
3. **Apple Books** (через Draft2Digital) — iPhone-аудитория Израиля/США.
4. **Физмагазины диаспоры** (Исрадон/Дон Кихот — Израиль; MnogoKnig/Муравей — Германия; Ruslania — Европа) — для PR-присутствия, малый объём.
- ❌ **LitRes** — оплата иностранными картами заблокирована; **Boosty/Prodamus** — те же проблемы РФ.

## WHITE SPACE — ЧЕМ «НОВЫЙ КОД» ОТЛИЧАЕТСЯ

| Критерий | Бронникова/Мотова/Донова | «Новый Код» |
|---|---|---|
| Личная история автора | Нет / частично | **Да — анорексия, депрессия, суицидальные кризисы** |
| Мужской голос | Нет (все авторы — женщины) | **Да — редчайший случай** |
| Трансформация близкого (дочь −50 кг) | Нет | **Да — реальный кейс, применимо и для семей** |
| Чёткая структура / система | Хаос (жалоба на Бронникову) | **5 установок + метафора код/антивирус** |
| Движение + психика как компоненты | Почти нет (фокус на еде) | **Да — интегрированная система** |
| Тон | «сверху вниз», экспертный | **На равных, изнутри опыта** |

**Вывод:** в русскоязычном поле «Новый Код» отличается даже сильнее, чем в англоязычном — потому что доминирующий конкурент (Бронникова) ровно по этим осям уязвим. Позиционирование против неё: «не эксперт сверху, а человек, прошедший это сам; не хаос идей, а система из 5 установок; не только еда, а психика + тело + движение».

## УВЕРЕННОСТЬ И ОГРАНИЧЕНИЯ

- **Высокая:** профили конкурентов, рейтинги, отзывы (LiveLib/LitRes/iRecommend/MyBook — прямой fetch); дистрибуция (технически подтверждено); платёжные барьеры LitRes.
- **Средняя:** численность диаспоры (порядок величины, расхождение методологий «этнические» vs «русскоязычные»); платёжеспособность сегментов; интент запросов (без Wordstat).
- **Нет данных:** объём продаж self-help в диаспоре; аудитория Telegram-каналов в цифрах; книжное потребление волны-2022.

## 📖 Расшифровка терминов

- **Диаспора** — здесь: русскоязычные за пределами РФ (Германия, Израиль, США, Канада, Прибалтика и др.).
- **РПП** — расстройство пищевого поведения; **интуитивное питание** — питание по сигналам голода/насыщения без диет.
- **VOC** — «голос клиента»: дословные отзывы аудитории.
- **White space** — незанятый рыночный пробел / пробелы конкурента (у Бронниковой — нет личной истории, нет мужского голоса и т.д.).
- **KDP (Kindle Direct Publishing)** — самиздат Amazon; **Gumroad / Payhip** — платформы прямых продаж цифровых товаров (Stripe/PayPal, работают для диаспоры); **Ridero** — сервис самиздата, выгружает на Amazon/Google Books.
- **LiveLib / LitRes / iRecommend / MyBook** — русскоязычные платформы книжных отзывов/продаж (источники рейтингов и цитат).
- **IntuEat** — центр/экосистема Светланы Бронниковой (книга + программы + Telegram-канал).
